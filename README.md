 Fuzzy Logic Washing Machine & Student Performance System

 Overview
This project consists of two parts:

 Q1: Fuzzy Logic Washing Machine
A fuzzy logic system designed using MATLAB to determine washing time based on:
- Dirt Level
- Load Size

 Q2: Student Performance Prediction
A simple hybrid intelligent system implemented in Python to predict student performance based on:
- Attendance
- Assignment Marks
- Test Marks

 Files Included
- washing_machine.fis
- screenshots.pdf
- Fuzzy_Washing_Machine_Report.pdf
- Q2_Student_Performance_Report.pdf
- student_performance.py
- screenshot pdf of q2

Description
The washing machine system uses Mamdani fuzzy inference to determine optimal wash time.

The student performance model uses rule-based logic inspired by fuzzy systems and basic neural decision behavior.
